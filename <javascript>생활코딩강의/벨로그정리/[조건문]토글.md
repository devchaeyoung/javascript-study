# 토글이 뭔가여?

- 토글스위치(toggle swich) : on/off하나밖에 없는 스위치
- 두가지 상태중 하나를 선택하는데 쓰는 키를 토글이라한다.
- 동사로는 (토글키로)껏다,켰다 할때 쓰인다.

# 토글 HTML로 구현해보자

- 아래처럼 비교연산자로 toggle을 활용할 수 있다.

```html
<title>토글(toggle)구현하기</title>
<h1>comparison operator & boolean</h1>

<h2>비교연산자(===)</h2>

<h3>1===1</h3>
<script>
  document.write(1 === 1);
</script>

<h3>1===2</h3>
<script>
  document.write(1 === 2);
</script>

<h3>1==="1"</h3>
<script>
  document.write(1 === "1");
</script>

<h2>1&lt;2</h2>
<script>
  document.write(1 < 2);
</script>
```
